# Убежище (прототип)

3D survival + tower-defense, вдохновлён режимом «Убежище» из CS Nexon: Zombies.
Движок: **Godot 4**. Полный план развития: [docs/ROADMAP.md](docs/ROADMAP.md).

## Как запустить (Шаг 1)

1. Скачай **Godot 4** (версия Standard, не .NET): https://godotengine.org/download
   Это один `.exe`, установка не нужна — просто запусти.
2. В окне Godot нажми **Import** (Импорт) → укажи файл `project.godot` из этой папки → **Import & Edit**.
3. Нажми **F5** (или кнопку ▶ «Play» вверху справа).
4. Готово: бегай капсулой по полю.

## Управление

| Клавиша | Действие |
|---------|----------|
| W A S D | Движение |
| Мышь    | Обзор |
| Space   | Прыжок |
| ЛКМ     | Выстрел (курсор захвачен) / захватить курсор |
| Esc     | Освободить курсор |

## Что внутри

- `project.godot` — настройки проекта.
- `scenes/main.tscn` — главная сцена (земля, свет, небо, игрок).
- `scenes/player.tscn` — игрок (капсула + камера от первого лица).
- `scripts/player.gd` — логика движения игрока.
