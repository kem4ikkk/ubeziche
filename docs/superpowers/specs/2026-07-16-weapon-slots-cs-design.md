# Дизайн: слоты оружия как в CS (Этап 4.44)

## Решение
Три слота (клавиши `1`/`2`/`3`), одновременно не больше одного оружия в слоте.
Покупка в занятый слот — замена (старое пропадает). Меню покупки у чёрного
рынка в стиле CS; рынок появляется только при `weapon_basic ≥ 1`.

## Слоты
| Слот | Клавиша | Содержимое |
|------|---------|------------|
| 1 Primary | `1` | Дробовик, автомат, снайперка |
| 2 Pistol | `2` | Пистолет, двойные пистолеты |
| 3 Melee | `3` | Топор / мачете / лом / молот |

- Старт: слот 2 = базовый пистолет, слот 3 = топор, слот 1 пуст; в руках — топор.
- `Q` больше не достаёт топор.
- Пустой слот 1 по `1` — ничего.
- Крафт мачете/лома/молота заменяет топор в слоте 3.

## Гейты навыков
- Чёрный рынок виден/активен только при `weapon_basic ≥ 1`.
- Двойные пистолеты: `weapon_basic ≥ 1`.
- Дробовик / снайперка: `weapon_mid ≥ 1`.
- Автомат: `weapon_adv ≥ 1`.
- Урон/скидка навыков «Мастер оружия» сохраняются.

## Покупка
CanvasLayer-меню у рынка (F рядом): категории + номера + цена.
Недоступные по навыку — тусклые + подсказка. Покупка → деньги → замена слота → экип.
